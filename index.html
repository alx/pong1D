PONG 1D

1D LED Ping Pong — Luminous table game powered by LEDs

// วิธีการเล่น

01
🕹️ ยืนที่ฝั่งของคุณ ผู้เล่นแต่ละคนยืนอยู่ที่ปลายแถบ LED ปุ่มของคุณอยู่ตรงหน้า
02
💡 จับตาดูลูกบอลแสง แสงสีขาวเดินทางตามแถบ LED จากผู้เล่นหนึ่งไปยังอีกคน — นั่นคือลูกบอล
03
👆 กดเมื่อถึงฝั่งคุณ กดปุ่มของคุณตอนที่ลูกบอลมาถึงฝั่งคุณเพื่อตีกลับ
04
⏱️ จังหวะคือทุกสิ่ง กดเร็วหรือช้าเกินไปและคู่ต่อสู้จะได้คะแนน
05
⚡ พลังพิเศษ — ความเร็วสูงสุด กดปุ่มของคุณเมื่อแสงอยู่ติดกับปุ่มเพื่อทริกเกอร์โบนัสความเร็ว — ลูกบอลพุ่งไปอีกฝั่ง!
06
🏆 คนแรกที่ได้ 5 คะแนนชนะ ทำคะแนนได้ 5 แต้มเพื่อชนะแมตช์ แถบ LED จะสว่างเป็นสีของคุณเพื่อแสดงชัยชนะ

// เกี่ยวกับ

Pong 1D นำเกม Pong คลาสสิกมาสร้างสรรค์ใหม่ในมิติเดียว — แถบ LED แทนที่หน้าจอ ผู้เล่นควบคุมแป้นตีตามแถบ ส่งลูกบอลแสงไปมาด้วยความเร็วที่เพิ่มขึ้น

ออกแบบสำหรับงานอีเว้นท์ แฮกกาธอน และงาน Maker Fair รองรับผู้เล่น 2-4 คนพร้อมกัน เกมเป็น Open Source และสามารถประกอบได้จากชิ้นส่วนทั่วไป

แสดงบน fauxdepart.com — ดูแกลเลอรีและเอกสารโครงการฉบับเต็ม
